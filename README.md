# Restaurant Sync

Минимальное веб-приложение для синхронизации официантов, кухни и руководства без внешних зависимостей.

## Что умеет

- Официант создает заказ из нескольких блюд, количества, цены и пожеланий.
- Официант выбирает блюда из справочника с поиском и может добавлять неограниченное число позиций в заказ.
- Кухня видит общую очередь заказов, берет заказ на себя и меняет статусы.
- Руководство видит общую панель, все заказы, автоматические строки отчетов, смены и сотрудников.
- Руководство управляет справочником блюд и цен, а время работы сотрудников автоматически учитывается от входа до выхода.
- Для каждого сотрудника считается персональная статистика: заказы, выручка, часы, смены и текущая зарплата.
- Руководство может изменять все данные и создавать новые учетные записи.
- Данные автоматически сохраняются в переносимую папку `data`, поэтому при копировании проекта на другой ПК можно продолжить работу с теми же записями.

## Запуск

```powershell
py app.py
```

После запуска приложение будет доступно по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

## Хранилище

При первом запуске автоматически создаются:

- `data/restaurant.db` — основная локальная база
- `data/restaurant_snapshot.json` — переносимый JSON-снимок

Если перенести проект вместе с папкой `data` на другой ПК, заказы, сотрудники, меню и отчеты сохранятся.
